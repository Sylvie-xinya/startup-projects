# 我们怎么做生意

## 核心理念

**不自己想点子，借别人的势**

看到一个火的工具/项目 → 别人已经在用、在搜 → 我们做个网站帮人找到它 → 别人点我们的链接买东西，我们拿佣金

## 为什么这样做

- GitHub 趋势项目 = 已经被市场验证的需求
- 已经有人在搜、在用、有流量
- 我们不需要自己创造需求，只需要接住流量

# 怎么赚钱

## 1. 推荐拿佣金（最快）
- 推荐 AI 工具、软件等产品

## 3. 卖模板/教程
- 做好的模板、教程，一次制作持续卖
- **睡后收入**

# 第三部分：怎么做

## 通用流程

```
选方向 → 买域名 → 建站 → 写内容 → SEO → 挂链接 → 等钱来
```

## 第1步：选方向

参考上面的项目列表，选一个感兴趣的。

## 第2步：买域名

- 推荐: Namecheap、GoDaddy
- 价格: $10-15/年
- 技巧: 和主题相关、简短好记

## 第3步：建网站

- **WordPress**: 最适合新手，生态成熟
- **Astro**: 适合懂技术的，速度快
- **GitHub Pages**: 完全免费

## 第4步：写内容

- 写 15-20 篇 SEO 文章
- 类型: 对比、教程、排行榜
- 用 AI 帮忙写

## 第5步：SEO 优化

- 标题包含关键词
- 提交 Google Search Console
- 外链: GitHub + 社媒

## 第6步：挂联盟链接

| 平台 | 佣金 |
|------|------|
| ElevenLabs | 30% |
| Loom | 30% |
| Riverside | 30% |
| Cursor | 佣金 |
