## 智能体管理

### ClawTeam ⭐最近很火
- **链接**: https://github.com/HKUDS/ClawTeam
- **是什么**: 一个界面同时管理多个 AI 智能体一起工作
- **怎么用**: 以前一个人干，现在可以让多个 AI 同时帮你写代码、查资料、做分析

## 工作流自动化

### n8n
- **链接**: https://github.com/n8n-io/n8n

# 💰 第二部分：商业项目

## 核心理念

**不自己想点子，借别人的势**

看到一个火的项目/工具 → 别人已经在用、在搜 → 我们做个网站帮人找到它 → 别人点我们的链接买东西，我们拿佣金

---

## 我们怎么赚钱

### 1. 推荐拿佣金
- 推荐 AI 工具、软件等
- 别人通过我们的链接买了 → 我们拿 20-30% 佣金
- 不用囤货、不用售后

### 2. 广告费
- 网站有流量后，广告商主动找上门

### 3. 卖模板/教程
- 一次制作持续卖，睡后收入

## ⚡ 怎么做

1. 选一个方向
2. 买域名（$10/年）
3. 做个简单的网站
4. 写 15-20 篇介绍工具的文章
5. 等 Google 收录
6. 挂上推荐链接，等钱来

## 📝 核心原则

- **不拍脑袋** - 看数据，不靠自己猜
- **先赚钱** - 先想好怎么变现，再做产品
- **快速试错** - 不追求完美，先跑通
- **一个跑通了就复制** - 成功了就复制到下一个方向
