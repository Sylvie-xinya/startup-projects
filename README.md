# 💡 创业项目库

这是我们团队的赚钱思路库，用大白话写的，方便每个人都能看懂。

---

## 🎯 我们怎么做生意

**不自己想点子，借别人的势**

看到一个火的项目/工具 → 别人已经在用、在搜了 → 我们做个网站/工具帮人找到它 → 别人点我们的链接买东西，我们拿佣金

---

## 💰 我们怎么赚钱

### 1. 推荐拿佣金
- 推荐 AI 工具、录屏软件等
- 别人通过我们的链接买了 → 我们拿 20-30% 佣金
- 不用囤货、不用售后

### 2. 广告费
- 网站有流量后，广告商主动找上门
- 工具类网站广告费比较高

### 3. 卖模板/教程
- 做好的模板、教程，一次制作持续卖
- 睡后收入

## ⚡ 怎么做

1. 选一个方向
2. 买域名（$10/年）
3. 做个简单的网站
4. 写 15-20 篇介绍工具的文章
5. 等 Google 收录
6. 挂上推荐链接，等钱来

## 📝 核心原则

- **不拍脑袋** - 看数据，不靠自己猜
- **先赚钱** - 先想好怎么变现，再做产品
- **快速试错** - 不追求完美，先跑通
- **一个跑通了就复制** - 成功了就复制到下一个方向
