# 我们怎么做生意

## 核心理念

**不自己想点子，借别人的势**

看到一个火的工具/项目 → 别人已经在用、在搜 → 我们做个网站帮人找到它 → 别人点我们的链接买东西，我们拿佣金

## 为什么这样做

- GitHub 趋势项目 = 已经被市场验证的需求
- 已经有人在搜、在用、有流量
- 我们不需要自己创造需求，只需要接住流量

## 我们怎么赚钱

### 1. 推荐拿佣金（最快）
- 推荐 AI 工具、软件等产品
- 别人通过我们的链接购买 → 我们拿 20-30% 佣金
- **不用囤货、不用售后**

### 2. 广告费
- 网站有流量后，广告商主动找上门
- 工具类网站广告费比较高

### 3. 卖模板/教程
- 做好的模板、教程，一次制作持续卖
- **睡后收入**

### 是什么
帮人找到最好的录屏软件，挂在我们的推荐链接上，别人点了我们拿佣金。

### GitHub 原始项目
https://github.com/siddharthvaddem/openscreen（MIT 协议）

### 赚钱原理
1. 做一个录屏工具测评网站
2. 对比各个工具的优缺点
3. 挂 Loom、Riverside 等的推荐链接
4. 别人通过我们的链接买了 → 拿 30% 佣金

### 具体步骤（超详细）

**第1步：买域名**
- 去 Namecheap 或 GoDaddy 买
- 推荐域名: `screenrecorderguide.com` 或类似

**第2步：建网站**
- 用 WordPress 或 Astro
- 选一个简洁的主题
- 不要花太多时间在设计上

**第3步：写内容（重点）**
- 写 15-20 篇文章

  - "如何用 OpenScreen 做产品演示视频"
  - "无水印录屏软件推荐"
- 每篇文章包含:
  - 工具介绍
  - 优缺点分析
  - 使用教程
  - 我们的推荐链接

**第4步：SEO 优化**
- 标题包含关键词: "best free screen recorder"
- 文章要有结构（小标题、列表）
- 图片加 alt 标签
- 内部链接

**第5步：提交 Google**
- 去 Google Search Console 提交网站
- 等 Google 收录（1-4周）

**第6步：挂联盟链接**
- 申请 Loom 联盟: https://www.loom.com/affiliates
- 申请 Riverside 联盟
- 把链接挂在对应工具下面

# 第三部分：怎么做（详细步骤）

## 通用流程

每个项目的做法都遵循这个流程：

```
选方向 → 买域名 → 建站 → 写内容 → SEO → 挂链接 → 等钱来
```

---

## 第1步：选方向

参考上面的项目列表，选一个感兴趣的。

选项目标准：
- 你自己对这个领域有了解吗？
- 有热情持续做下去吗？
- 联盟佣金高吗？

---

## 第2步：买域名

推荐网站：
- Namecheap.com
- GoDaddy.com
- Cloudflare Registrar

域名选择技巧：
- 和你的主题相关
- 简短好记
- 包含关键词（如 screen、voice、resume）

价格：$10-15/年

---

## 第3步：建网站

推荐工具：

### WordPress（最适合新手）
- 主题: Astra、GeneratePress
- 页面构建器: Elementor
- 优点: 生态成熟，插件多

### Astro（适合懂技术的人）
- 静态网站，速度快
- SEO 友好
- 免费托管在 Cloudflare Pages

### GitHub Pages（最简单）
- 完全免费
- 需要会一点代码
- 适合个人博客

---

## 第4步：写内容（核心）

### 写什么

每篇文章围绕一个关键词：

| 文章类型 | 关键词例子 |
|----------|------------|
| 工具对比 | "Loom vs Riverside" |
| 排行榜 | "最佳录屏软件 TOP 10" |
| 教程 | "如何使用 OpenScreen" |
| 场景 | "游戏录屏软件推荐" |

### 怎么写（AI 帮忙）

1. 确定标题（包含关键词）
2. 用 AI 写大纲
3. 用 AI 扩写每个部分
4. 人工审核修改
5. 添加图片和内部链接

### 写多少篇

- 先做 15-20 篇
- 每周更新 1-2 篇
- 保持更新

---

## 第5步：SEO 优化

### 基础 SEO
- 标题包含关键词
- URL 简短（如 /best-screen-recorder）
- 图片加 alt 标签
- 有内部链接

### 提交 Google
1. 去 Google Search Console
2. 添加网站
3. 提交 sitemap
4. 等 Google 收录

### 外链
- 在 GitHub 项目页留言
- 社交媒体分享
- 相关论坛留言
