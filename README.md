## 🔌 浏览器插件方向

### 为什么做插件
- 开发周期短（1-2周）
- 用户获取成本低（Chrome Web Store）
- 可以爬取任意网页数据
- 直接向用户推送内容

### 变现方式
- **广告拦截器** → 付费去广告
- **SEO工具** → 订阅制
- **社交媒体管理** → 批量发帖
- **价格监控** → Affiliate 佣金

### 开发工具
- WXT (Next.js 风格)
- Plasmo (React 优先)
- vanilla JS (最轻量)

## 📝 商业原则

- **不靠感觉，靠数据** - GitHub 趋势 = 需求验证
- **先变现，后产品** - 流量来了再考虑自己做工具
- **快速 MVP** - 不完美，先跑通再说
- **批量复制** - 一个跑通了就复制到下一个 Niche
