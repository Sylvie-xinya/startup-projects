## openscreen - 录屏工具站

**数据来源**: GitHubWeekly 第110期  
**原项目**: https://github.com/siddharthvaddem/openscreen  
**Stars**: 爆发型增长  
**语言**: Electron + PixiJS  
**协议**: MIT

### 是什么
免费录屏工具的开源项目，用户多、话题热。

### 变现方式
1. **Google AdSense** - 工具类站广告 CPM 高
2. **联盟佣金** - 推广 Loom、Riverside 等付费工具拿佣金 30%
3. **付费合集** - "最佳录屏工具合集" PDF 卖 $9.9

### SEO 关键词
- best free screen recorder
- screen recording software
- record screen on mac/windows
- screen recorder no watermark

### 怎么做
1. 买域名（如 `screenrecorderguide.com`）
2. 用 WordPress/Astro 建站
3. 写 15-20 篇 SEO 文章
4. 挂 Loom/Riverside 联盟链接
5. 申请 AdSense

### 预计投入
- 域名: $10-15/年
- 主机: Cloudflare Pages (免费)
- 内容: AI 生成，30分钟/篇
