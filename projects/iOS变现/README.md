- 解锁详细性格分析 $2.99
- 解锁与其他性格的对比 $1.99

#### 3. 社交传播（免费增长）
- 一键分享到小红书、朋友圈、微博
- 生成好看的分享卡片
- 用户自发传播 = 免费获客

## AdMob 接入步骤

1. 创建 Google AdMob 账号
2. 添加 App
3. 获取 App ID 和广告单元 ID
4. 在代码中接入 AdMob SDK
5. 申请 Apple Developer 账号 ($99/年)
6. 提交审核

### 广告类型
- Banner: 底部小横幅，持久展示
- Interstitial: 全屏插屏，点击时展示
- Rewarded: 激励广告，看完给奖励
- Native: 原生广告，和内容融合
